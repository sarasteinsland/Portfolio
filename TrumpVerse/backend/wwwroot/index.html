TrumpMerches API Dokumentasjon

Velkommen til TrumpMerchandise API. Dette API-et gir tilgang til et utvalg av Trump-merchandise som T-skjorter, kopper, hatter og mer. Du kan hente informasjon om forskjellige produkter og filtrere etter type.

Base URL

Alle API-endepunktene kan nås på følgende base-URL:

http://localhost:5236/api/TrumpMerchandise

Endepunkter

1. Hent alle produkter

URL /
Metode GET
Beskrivelse Hent en liste over alle tilgjengelige Trump-merchandise.
Respons 
{
  "info": {
    "count": 3
  },
  "results": [
    {
      "id": 1,
      "name": "Trump T-shirt",
      "type": "T-shirt",
      "price": 25.99,
      "image": "shirt.jpg"
    },
    {
      "id": 2,
      "name": "Trump Mug",
      "type": "Mug",
      "price": 15.99,
      "image": "mug.jpg"
    },
    {
      "id": 3,
      "name": "Trump Hat",
      "type": "Hat",
      "price": 19.99,
      "image": "hat.jpg"
    }
  ]
}

2. Hent spesifikt produkt etter ID

URL /api/TrumpMerchandise/{id}
Metode GET
Beskrivelse Hent informasjon om et spesifikt produkt basert på produkt-ID.
Eksempel URL http://localhost:5236/api/TrumpMerchandise/1
Respons 
{
  "id": 1,
  "name": "Trump T-shirt",
  "type": "T-shirt",
  "price": 25.99,
  "image": "shirt.jpg"
}

3. Hent produkter etter type

URL /name/{name}
Metode GET
Beskrivelse Hent en liste over produkter som tilhører en spesifikk type (f.eks. "T-shirt", "Mug", etc.).
Eksempel URL http://localhost:5236/api/TrumpMerchandise/name/T-shirt
Respons 
{
  "info": {
    "count": 1
  },
  "results": [
    {
      "id": 1,
      "name": "Trump T-shirt",
      "description": "A bold T-shirt featuring the Trump logo.",
      "price": 19.0,
      "image": "trump_tshirt.png"
    }
  ]
}

Feilmeldinger

404 - Ikke funnet: Hvis produktet eller typen du søker etter ikke finnes.

400 - Dårlig forespørsel: Hvis forespørselen er feilformatert (f.eks. ugyldig produkt-ID).

Eksempler

Hent alle produkter:

curl -X GET http://localhost:5236/api/TrumpMerchandise

Hent produkt etter ID:

curl -X GET http://localhost:5236/api/TrumpMerchandise/1

Hent produkter etter type:

curl -X GET http://localhost:5236/api/TrumpMerchandise/name/T-shirt

Statuskoder

200 OK: Når forespørselen er vellykket.

404 Not Found: Når forespurte data ikke finnes.

400 Bad Request: Når forespørselen er feil.
